# Agent Instructions - Beverage Logistics Oman Website

Welcome, Agent! You are tasked with building a premium, enterprise-grade beverage logistics website for the largest beverage transportation and distribution company in Oman.

This project is planned to be executed in strict stages to ensure top-tier, $100k creative agency-level quality. 

---

## 🚀 The Master Roadmap

Follow the stages sequentially. Do not skip or combine stages unless explicitly instructed by the user.

1. **Stage 0: Creative Agency** (Brand Strategy, Personality, Color Palette, Typography, Complete Sitemap, Wireframes - NO CODE)
2. **Stage 1: Asset Production** (Hero images, warehouses, fleet, operations center images using high-end AI generator sites)
3. **Stage 2: UX Architecture** (Page-by-page user/business goals, section hierarchy, bottle journey scroll checkpoints, mobile fallback strategy)
4. **Stage 3: Design System** (Typography scale, grid system, spacing system, color tokens, motion rules - NO CODE)
5. **Stage 4: Build Enterprise Website** (Next.js 15, TypeScript, Tailwind CSS, Framer Motion - NO Three.js yet)
6. **Stage 5: Embed Three.js Storytelling Island** (React Three Fiber, Drei, GSAP ScrollTrigger - isolated WebGL island)
7. **Stage 5B: Mobile Fallbacks** (Width < 768px fallbacks, prefers-reduced-motion, non-WebGL fallbacks)
8. **Stage 6: Micro Interactions** (Reveal animations, hover effects, KPI counters, route line animations)
9. **Stage 7: Performance Optimization** (Lighthouse 95+, Draco compression, memory leak prevention)
10. **Stage 8: SEO & Launch** (Local SEO for Oman, Schema markup, sitemaps, Robots.txt)

---

## 🛠️ Tech Stack & Requirements

- **Framework**: Next.js 15 (App Router, React 19 compatible)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion & GSAP
- **3D/WebGL**: Three.js, React Three Fiber (R3F), `@react-three/drei`
- **Source Control**: Git. Remote origin: `https://github.com/Ameenrehman/logistic-oman.git`

---

## 📐 Styling & Design Philosophy

- **Quality Level**: Apple-level polish, Stripe-level clarity, DHL-level operational credibility.
- **Color Palette**: Deep, sophisticated corporate colors combined with warm Omani desert tones (golden hour lighting) and premium beverage brand accents.
- **Typography**: Modern, readable sans-serif typography (e.g., Google Fonts like Inter or Outfit) with strict scale constraints.
- **WebGL Balance**: WebGL is used as an **isolated storytelling island** in a specific scroll-controlled section, NOT as the background for the entire page. It must not block main thread page interactions.

## 🤖 Rules for AI Agents

1. **Read plan.md**: Always consult the main `plan.md` file for exact prompts and requirements.
2. **Stage Isolation**: Complete each stage fully before moving to the next. Verify the current stage with the user.
3. **No Placeholders**: When images or media are needed, use generated high-quality assets or exact SVG graphics. Never leave dummy boxes or blank images.
4. **Git Etiquette**: Commit changes after each successful stage completion with clear, descriptive commit messages.
